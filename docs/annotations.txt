Important functions:

Firebase.begin(HOST, CHAVE_SECRETA); - to begin a connection
Firebase.setInt("caminho", Valor Int) - to set int
Firebase.setFloat("caminho", Valor float) - to set float
Firebase.setString("caminho", Valor String) - to set string
Firebase.setBool("caminho", Valor Bool) - to set a Bool
Firebase.getInt("caminho") - to get a int
Firebase.getFloat("caminho") - to get a float
Firebase.getString("caminho") - to get a string
Firebase.getBool("caminho") - to get a bool
